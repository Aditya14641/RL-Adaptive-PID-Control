# Reinforcement learning based adaptive PID controller design for control of linear/nonlinear unstable processes

> T Shuprajhaa, Shiva Kanth Sujit, K. Srinivasan

To reproduce the results of the first order system and second order system, use the `scripts/run.sh`.

The systems are defined in `cs1.py` and `cs3.py`. These can be modified to evalute on other systems, specifically the transfer functions, which are defined at the top of the file.

The saved model weights as well as outputs are in the `data/logs` along with model outputs and other analyses. The run script reproduces the results given in that folder.
